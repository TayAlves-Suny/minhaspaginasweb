Meu contato com computação e programação começou ainda na graduação em Física, onde desenvolvi aplicações em Python voltadas à análise e visualização de dados experimentais. No mestrado, aprofundei essas habilidades ao criar um programa de Web Scraping para coleta e análise de dados de plataformas científicas, o que me proporcionou domínio de lógica de programação, estrutura de scripts, organização de dados e automação de processos.Essa experiência me permite transitar com segurança entre ambientes de desenvolvimento educacional e tecnológico, especialmente quando o objetivo é criar soluções digitais voltadas ao ensino de Ciências. No contexto do projeto para o Espaço CRIA, estou apta a contribuir com a prototipagem de aplicativos, elaboração de roteiros interativos, definição de fluxos de navegação pedagógica, e curadoria de dados educacionais — tudo isso com base em metodologias que envolvem o estudante como construtor do conhecimento.
